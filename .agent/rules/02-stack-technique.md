# Stack Technique

## Backend
- **Framework** : Laravel 11.x
- **Langage** : PHP 8.2+
- **Architecture** : Modulaire (Dossier `modules/`).
- **Base de Données** : MySQL .
- **Authentification / Permissions** : `spatie/laravel-permission`.

## Frontend
- **Build Tool** : Vite.
- **CSS Framework** : Botstrap.
- **UI Components** : AdminLTE 3 (`admin-lte`), Bootstrap Steps, Select2, Flatpickr.
- **Bibliothèques JS** : PrismJS, CodeJar, SweetAlert2.

## Testing
- **Unit / Feature** : PHPUnit 11.x.
- **Browser** : Laravel Dusk.

## Outils de Développement
- **Qualité du Code** : Laravel Pint.
- **Debug** : Laravel Debugbar.
